# DataViz-P4
Pair project for data visualization.

## Structure

1. pre-processing.ipynb: includes information about the datasets
2. feature_reduction.ipynb: includes the results + 2d/3d interactive figures of applying the feature reduction techniques.
3. figures: includes all of the figures generated by the notebooks
4. DigitsDataset: includes the digits recognizer dataset files
5. DryBeanDataset: includes the beans dataset files
